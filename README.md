# 📌 Портфолио "Обо мне" (Python Console Project)

## 👤 О проекте
Это консольное приложение-портфолио, созданное на Python с использованием ООП.

Проект показывает информацию обо мне в виде удобного меню с разделами.

---

## 🚀 Возможности проекта

- 📌 Консольное меню (8 разделов)
- 🧠 Использование ООП (класс Portfolio)
- 📂 Хранение данных в JSON-файле
- 🎨 Разделение логики и данных
- 📈 Прогресс-бар (визуализация "Точка А → Точка Б")
- 🚀 Экран загрузки при запуске

---

## 📂 Разделы

1. 👤 О себе  
2. 🎯 Моя цель  
3. 📖 История в IT  
4. 🧑‍🏫 Мой ментор  
5. 📈 Прогресс  
6. 🎮 Хобби  
7. 💼 Мои работы  
8. 🔗 GitHub  
9. 🚀 Прогресс-бар (визуальный)

---

## 🧠 Используемые технологии

- Python 🐍  
- ООП (классы)  
- JSON  
- Консольный интерфейс  
- Работа с файлами

1. Установить Python 3.10+

3. Запустить:

```bash
python main.py
