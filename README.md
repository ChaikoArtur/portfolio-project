# 📌 Портфолио "Обо мне" (Python Console Project)

## 👤 О проекте
Это консольное приложение, которое показывает моё портфолио в виде меню.  
Проект написан на Python с использованием ООП и работы с JSON.

---

## 🚀 Как запустить

1. Установить Python 3.10+
2. Открыть проект в PyCharm или терминале
3. Запустить файл:

```bash
python main.py
